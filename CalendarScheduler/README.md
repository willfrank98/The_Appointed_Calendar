The Appointed: A Calendar/Day Planner Web Application

The Appointed is a fullcalendar based web application that mostly shows use in forms, AJAX requests, CRUD elements and the ability to adapt and use libraries.

The external libraries used are as follows: 

Fullcalendar (Base Structure)
https://fullcalendar.io/

Flatpickr (Date/Time Picker) 
https://flatpickr.js.org/

Momentjs (Date Format for Javascript (Never use Javascript without Moment))
https://momentjs.com/

Spectrum (Color Picker)
https://bgrins.github.io/spectrum/
